📊 Indian Kids Screen Time Analysis
🚀 Project Overview

This project analyzes screen time usage among Indian children using Power BI. The goal is to uncover patterns in device usage, identify high-risk screen behavior, and compare trends across different demographics such as age groups and location (Urban vs Rural).

🎯 Objectives

Analyze average screen time across different age groups

Understand device usage distribution

Compare Urban vs Rural screen habits

Identify the percentage of children exceeding recommended screen limits

Visualize patterns using interactive dashboards

📂 Dataset

The dataset contains information about:

Age of children

Age Band classification (3–5, 6–9, 10–13, 14–17)

Average daily screen time (in hours)

Device type (Mobile, Tablet, Laptop, etc.)

Location (Urban/Rural)

Screen limit status (Exceeded / Not Exceeded)

Health risk category (Low, Moderate, High)

🛠️ Tools & Technologies

Power BI Desktop

Data Cleaning: Power Query

Data Modeling: DAX (Data Analysis Expressions)

📊 Key Visualizations
1. Average Screen Time by Age Band

Bar chart showing how screen usage varies across age groups

Helps identify high usage segments

2. Device Usage Distribution

Displays most commonly used devices

Useful for understanding digital behavior

3. Urban vs Rural Comparison

Compares average screen time between locations

Highlights lifestyle differences

4. Percentage Exceeding Screen Limit

Shows proportion of children exceeding safe screen limits

Indicates potential health risks

5. Age Band × Device Heatmap (Optional)

Matrix visualization showing device preference by age

Uses conditional formatting for better insights

🧠 Key Insights

Older age groups tend to have higher screen time usage

Mobile devices dominate usage among all age groups

Urban children generally show higher screen exposure

A significant percentage of children exceed recommended limits, indicating health concerns

⚠️ Challenges Faced

Handling incorrect aggregation leading to 100% exceedance issue

Data cleaning and proper categorization of age groups

Managing filter context in DAX measures

✅ Solutions Implemented

Used FILTER() function in DAX to correctly calculate percentages

Verified data distribution to ensure accuracy

Applied proper data modeling techniques
